## Sistema de Información para la Gestión de Transporte Especial de Pasajeros

1. Descripción del proyecto

El proyecto consiste en el desarrollo de un sistema de información web para la empresa Ultra Servicio Especial de Pasajeros, con el objetivo de mejorar la organización y administración de los servicios de transporte.

El sistema permitirá gestionar los procesos principales de la empresa como el registro de usuarios, autenticación, consulta de rutas, creación de reservas, asignación de servicios y administración de la información.

La plataforma busca reemplazar procesos manuales por una solución digital que facilite el acceso a la información, mejore la atención al cliente y permita una gestión más eficiente de los recursos de la empresa.

2. Problemática identificada

Actualmente, la gestión del transporte especial puede presentar dificultades relacionadas con la organización de usuarios, control de reservas, manejo de rutas y almacenamiento de información.

La falta de una plataforma centralizada puede ocasionar pérdida de información, retrasos en los procesos y dificultades para consultar datos importantes del servicio.

Por esta razón, se plantea desarrollar un sistema que permita automatizar estos procesos y mejorar la comunicación entre clientes y empresa.

3. Objetivo general

Desarrollar un sistema de información web para Ultra Servicio Especial de Pasajeros que permita administrar usuarios, rutas, reservas y servicios de transporte, mejorando la organización de la información y optimizando la prestación del servicio.

4. Objetivos específicos
Diseñar una interfaz web sencilla y fácil de utilizar para los usuarios.
Implementar un módulo de registro e inicio de sesión.
Crear un sistema para gestionar rutas y servicios disponibles.
Permitir la creación y consulta de reservas de transporte.
Organizar la información mediante una base de datos.
Facilitar la administración del sistema mediante roles y permisos.
5. Usuarios del sistema
Cliente

Usuario que requiere utilizar el servicio de transporte.

Funciones principales:

Registrarse en la plataforma.
Iniciar sesión.
Consultar rutas disponibles.
Realizar reservas.
Consultar sus servicios solicitados.
Administrador

Persona encargada de controlar y gestionar la información del sistema.

Funciones principales:

Administrar usuarios.
Gestionar rutas.
Controlar reservas.
Administrar roles y permisos.
Actualizar información del sistema.
Personal operativo

Usuarios encargados de apoyar la operación del servicio.

Funciones principales:

Consultar servicios asignados.
Revisar información de rutas.
Gestionar información relacionada con los viajes.
